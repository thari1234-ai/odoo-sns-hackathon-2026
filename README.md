GlobeTrotter - Travel Planning App
Overview
GlobeTrotter is an intuitive and user-friendly travel planning web application designed to help users create, manage, and share detailed trip itineraries. Whether planning a solo adventure or a group vacation, GlobeTrotter streamlines itinerary building, budget tracking, and activity scheduling in one centralized platform.
________________________________________
Features
•	Trip Creation: Easily create and customize trips with destinations, dates, and travel details.
•	Itinerary Builder: Add and organize stops, activities, and events in a clear timeline.
•	Budget & Cost Breakdown: Track trip expenses and analyze spending with a detailed budget dashboard.
•	User Profiles: Manage personal travel preferences, saved trips, and settings.
•	Public Sharing: Share itineraries publicly with friends or the community.
•	Activity Search: Discover activities and places to visit within your destinations.
•	Calendar Integration: Visualize your trip plans on a timeline calendar for easy tracking.
•	Responsive Design: Works seamlessly across devices for travel planning on-the-go.

Tech Stack

Setup & Installation
1.	Clone the repository:
2.	git clone https://github.com/your-repo/travel-planner.git
3.	cd travel-planner
4.	Install dependencies (if any):
This is primarily a static frontend app integrated with Supabase; if any Node.js or package management is used, list commands here.
5.	Configure Supabase:
o	Create a Supabase project at supabase.io
o	Copy your API keys and update relevant config files (e.g., auth.js, environment variables)
o	Ensure database tables match the expected schema for trips, itineraries, users, etc.
6.	Run the app:
o	Open the .html files directly in a browser or serve via a local server:
7.	npx serve .
